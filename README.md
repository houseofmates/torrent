<h1 align="center">torrent client</h1>

this is a vibecoded torrent client built with svelte and vite. it provides a web-based interface for managing torrent downloads.

features:
- intuitive web ui
- integration with qbittorrent api
- real-time updates

to run locally:
1. install dependencies: npm install
2. start development server: npm run dev --port 3004 --host
3. open http://localhost:3004

the service automatically starts on port 3004 across reboots.
